# iPhone 15 Pro — Product Showcase Page

This project contains the source code of a modern landing/view page created for the iPhone 15 Pro. The design is fully built on static HTML/CSS/JS.

---

## Features

- Single-page layout highlighting the design and technical details of the iPhone 15 Pro.
- High-quality images, video background, and animations.
- Fully responsive layout.
- Dark theme enabled by default (`data-bs-theme="dark"`).
- Scroll-triggered animations, interactive slider components, and modal video playback.
- Semantic HTML structure.

---

## Technologies Used

### Core
- **HTML5**
- **CSS3**
- **JavaScript**

### Styling & Layout
- **Bootstrap 5**  
  Used for grid layout, utility classes, and responsive structure.
- **Custom CSS (main.css)**  
  Responsible for custom animations, theme colors, typography, and section layouts.

### Animation & Interactivity
- **AOS (Animate on Scroll)** — scroll-triggered animations  
- **Swiper.js** — product/gallery slider  
- **GLightbox** — video and image lightbox system
